# WP Dash Standalone

:::danger Highly Experimental
This plugin is in **early development** and is **not recommended for production use**.

- It may be unstable or break your WordPress admin
- It is **not guaranteed to work** with all WordPress configurations
- APIs and behavior may change at any time without notice
- No support is provided for issues caused by this plugin

**Use at your own risk.**
:::

## What Is It?

WP Dash Standalone embeds the WP Dash management interface directly inside the WordPress admin panel, eliminating the need for a separate hosted dashboard.

**Version:** 1.0.0 *(experimental)*  
**License:** GPL-2.0-or-later  
**Requires:** WordPress 5.6+, PHP 7.4+

## Download

- [`wp-wpdash-plugin.zip`](https://github.com/giulio-leone/wpdash/raw/main/wp-wpdash-plugin.zip) ⚠️

## Installation

1. Go to **WordPress Admin → Plugins → Add New → Upload Plugin**
2. Upload `wp-wpdash-plugin.zip`
3. Click **Activate Plugin**

:::tip Recommended Alternative
For a stable experience, use the **[WP Dash Bridge](./wp-bridge)** plugin instead, paired with the hosted or self-hosted WP Dash dashboard.
:::
